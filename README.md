# nishi327.gihub.io

このリポジトリはシンプルなランディングページを収めています。`index.html` をブラウザで開くと、
グラデーションと光の演出を取り入れた「かっこいい」ホームページが表示されます。

## セットアップ
- `index.html` を直接開けば完成です（ビルド不要）。
- 必要であれば、静的ホスティングサービスにそのままデプロイできます。
